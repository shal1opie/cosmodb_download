Общее описание

Проект представляет из себя веб-интерфейс фактографической базы данных космических достижений СССР и России, релизованный в виде одностраничного приложения на языке PHP. Цель проекта заключена в предоставлении пользователям ресурса для информационной, справочной и методической работы, а также повышения осведомленности о космических достижениях прошлого и настоящего.

Требования

Системные требования
Процессор Совместимый, тактовая частота не ниже 500 MHz
ОЗУ не менее 512 Мб
ПЗУ Не менее 10 ГБ свободного дискового пространства

Установка

Перед загрузкой, убедитесь что у вас установлен дистрибутив для сборки локального веб-сервера с версией php 5.6 или выше. Загрузите архив на сайте или с репозитория Github по ссылке https://github.com/shal1opie/cosmodb_stable. Переместите папку cosmodb в рабочую папку дистрибутива, например для XAMPP - htdocs. Перейдите в поисковой стркое браузера по адресу: localhost/cosmodb/main/cosmodb.php.

Начало работы

При первом запуске системы программа автоматически создаст базу данных и перейдет на форму регистрации администратора. Поля формы регистрации уже заполнены данными, при необходимости их следует изменить. Так же имеется возможность просмотра пароля. Введенные данные следует запомнить удобным для вас способом для дальнейшей успешной авторизации в систему. После регистрации вы сможете авторизироваться.
