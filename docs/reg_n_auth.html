Создание нового аккаунта

Для перехода на форму регистрации необходимо в форме авторизации нажать на текст "Зарегистрироваться".

После чего необходимо заполнить поля формы данными для регистрации согласно их описанию.

После регистрации система перенаправит вас на форму авторизации.

Вход в существующий аккаунт

Для входа в систему необходимо заполнить поля формы согласно их описанию.

При неверном вводе данных система выведет сообщение об ошибке ниже полей формы.

После успешной авторизации система отобразит страницу со статьями, в случае если ваша роль - пользователь. В ином случае система отобразит страницу работы с базой данных.
